  • ¿Como se instalan y desistalan modulos?

  • Cómo personalizar la visualización

    • Esto sirve para cambiar el tema de Eclipse, primero vamos a Windows y luego a preferences. Despues, iremos a general, preferences, y veremos que hay un apartado donde pone temas y veremos que hay diferentes temas, elegimos el que queramos y luego le daremos a Apply and Close.

  • Automatizaciones de acciones / Atajos

    Estos son los mas basicos, si quires saber mas Pincha aqui.

    1. Buscar en todos el programa algún comando --> mayus + mayus

    2. Ctrl + Shift + R --> Abrir cualquier archivo rápidamente

    3. Ctrl + Shift + T --> Abrir cualquier clase del proyecto

    4. F3 --> Ir a la definición de una función o variable

    5. Ctrl + Shift + F --> formatear código (tabulaciones, saltos de línea,…)

    6. Ctrl + Shift + C --> comentar/descomentar con // las líneas seleccionadas

  • Actualizaciones

    En Eclipse seria darle a help --> Chek for Updates, y luego reiniciar la aplicacion. Esto permite mantener actualizado tu entorno de desarrollo Eclipse y sus componentes.
